About

Flower Classifier App is a demo of an image recognition neural network that classifies types of flowers.

Summary

This is a demo of an image recognition machine learning app that identifies flower classes from an image. The image datasets are from http://www.robots.ox.ac.uk/~vgg/data/flowers/102/index.html.

See the neural network in action on a random sampling of flower images here.

Network Architecture

The example neural network in this demo was written and trained with PyTorch (see the repo where this network was trained). The network is based off the VGG13 model architecture. Hyperparameters used in training:

  • Learning Rate: 0.0001
  • Dropout Rate: 0.2
  • Classification Input Size: 25088
  • Classification Hidden Layers (1): 12544
  • Classification Output Size: 102
  • Criterion: nn.NLLLoss()
  • Epochs Trained: 6

Training & Testing Results

During training, the network rapidly increased the correct prediction accuracy. It seemed to peek at around 6 epochs and 92-94% accuracy on the test dataset.
